---
title: "Get Connector"
description: "Fetch metadata and status for a single connector."
openapi: "api-reference/v2/openapi.json GET /connectors/{id}"
---

<RequestExample>

```bash cURL
curl 'https://api.hydradb.com/connectors/{connector_id}' \
  -H "Authorization: Bearer $HYDRA_DB_API_KEY" \
  -H "API-Version: 2"
```

</RequestExample>

## Path parameters

| Name | Description |
| --- | --- |
| `id` | Connector UUID returned by `POST /connectors`. |

<ResponseExample>

```json 200
{
  "connector_id": "{connector_id}",
  "provider": "slack",
  "name": "acme-engineering",
  "tenant_id": "acme_corp",
  "sub_tenant_id": "engineering",
  "status": "active",
  "created_at": "2026-06-01T12:00:00Z",
  "next_sync_at": "2026-06-01T13:00:00Z"
}
```

</ResponseExample>
